# Interactive Map of African Restaurants in the U.S.

## Overview
This project is an interactive web map that displays African restaurants across the United States using the ArcGIS API for JavaScript.

The map allows users to explore restaurant locations, view details, and navigate using an interactive basemap.

---

## Objective
- Build a web-based GIS application
- Visualize African restaurant locations across the U.S.
- Demonstrate web mapping using ArcGIS API for JavaScript

---

## Tools & Technologies
- ArcGIS API for JavaScript (4.x)
- HTML
- CSS
- JavaScript
- CSV Data

---

## Data
- Dataset: African restaurants (CSV format)
- Fields include:
  - Name
  - City
  - State
  - Description
  - Latitude & Longitude

---

## Features
- Interactive web map
- Zoom and pan functionality
- Custom restaurant icons (fork & knife)
- Popups showing restaurant details
- Legend for map interpretation
- Clean street basemap

---

## Methodology
1. Prepared restaurant dataset in CSV format
2. Loaded data using CSVLayer in ArcGIS API
3. Applied custom symbology using picture markers
4. Configured popup templates for restaurant details
5. Added legend widget for map interpretation
6. Built full web map using HTML, CSS, and JavaScript

---

## Workflow

CSV Data → ArcGIS API (CSVLayer) → Map Rendering → Symbolization → Popups → Interactive Web Map


---

## Results
- Fully interactive web-based map
- Users can:
  - Click on restaurants to view details
  - Zoom and explore locations
  - Navigate using a street basemap
